3. ¿Qué etiqueta se utiliza para insertar código JavaScript en HTML?

Se usa la etiqueta <script> </script>

4. ¿Qué ventajas tiene incrustar el código JavaScript en un archivo externo?

Se consigue una separación entre el código y la estructura de la página, además de facilitar compartir comnpartir el código entre disitintas páginas, centralizar el código y rapidez de carga

5. ¿Qué extensión suelen tener los archivos que contienen código JavaScript?

Tienen la extensión .js

6. ¿Cómo se referencia un archivo .js en HTML?

Usando una etiqueta <script> y dándole como valor al atributo src el nombredel fichero que contiene el código JavaScript. Ej: <script src=codigoJS.js></script>

7. ¿Qué debe hacer si el archivo JavaScript se encuentra en otro directorio?

Igual que el caso anterior, pero habría que especificar la ruta (ya sea absoluta o relativa) de la ubicación del fichero. Ej: <script src=./Documentos/ejercicios/codigoJS.js></script>

8. ¿Qué etiqueta se usa para mostrar un mensaje si el navegador no soporta JavaScript?

Para ello se usa la etiqueta <noscript> <noscript>

10. ¿Qué es la ofuscación de código y por qué se utiliza?

La ofuscación de código, es una técnica empleada para proteger el código de fuente de una página contra fraudes u otros visitantes. Incluye prácticas como eliminación de saltos de línea, espacios en blanco inncesarios, tabuladores, nombres intelegibles en funciones y variables, variables de almacenaje de código, recursividad, etc.

12. ¿Qué nombre le dio Microsoft a JavaScript?

13. ¿Por qué Microsoft usó el nombre Jscript?

Para evitar problemas relacionados con la marca

14. ¿Qué problema surgió con la versión de JScript de Microsoft?

Surgieron problemas por incompatibilidades que su versión de JavaScript tenía con múltiples navegadores.

15. ¿Qué es el DOM y por qué fue diseñado?

El DOM (Modelo de Objeto del Documento), es una interfaz de programación que permite a los navegadores y al código interactuar con la estructura, contenido y el sitio de un documento HTML o XML.Fue diseñado para solucionar las incompatibilidades de JScript con los navegadores web

16. ¿Qué navegadores incorporaron el DOM desde su primera versión?

Internet Explorer 6, Netscape Navigator, Opera 7 y Mozilla Firefox

17. ¿Qué estándar regula el lenguaje JavaScript desde 1997?

El estándar ECMA

18. ¿Qué documento detalla las especificaciones del estándar ECMA?
